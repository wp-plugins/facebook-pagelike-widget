== Description ==

espn.com website is the most popular website for daily sport news and events. It also provide API to list their news and updates into your webpage. This plugin will display latest news and all sports related upgates from espn.com. As an admin you need to create one account at http://developer.espn.com/member/register and after registration you will get one API key. Use this api key in the admin while configure widget. In the first version it will display some news & will update soon once get response from users.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
